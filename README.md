# 🧱 可爱版俄罗斯方块

给小朋友做的网页俄罗斯方块小游戏 —— 单文件、零依赖，打开即玩。

## 🎮 在线试玩

**https://paultunggm-pixel.github.io/tetris/**

## ✨ 特色

- 12×22 大棋盘，7 种经典方块（7-bag 发牌，分布均匀）
- 🎨 圆角渐变粉彩方块，下落方块带小眼睛
- 🎉 消行鼓励语动画（"好耶！"、"太棒了！"、"四消无敌！"）
- 🎵 WebAudio 合成经典背景音乐 + 音效（🔊 可开关，无需任何音频文件）
- 📱 自动适配 PC / iPad / 手机，触屏设备显示触控按钮

## ⌨️ 操作

| 按键 | 动作 |
|------|------|
| ← → | 移动 |
| ↓ | 软降（每格 +1 分） |
| ↑ / X | 顺时针旋转 |
| Z | 逆时针旋转 |
| 空格 | 硬降（每格 +2 分） |
| P | 暂停 / 继续 |
| R | 重新开始 |

## 🛠 技术

纯 HTML + CSS + JS（Canvas 2D + WebAudio），无任何外部依赖。
本地试玩：直接用浏览器打开 `index.html` 即可。
